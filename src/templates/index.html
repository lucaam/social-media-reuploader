Access with admin token

Set the ADMIN_TOKEN environment variable before starting the GUI. You can access the admin endpoints using either:

  • Query parameter: /requests?token=YOUR_ADMIN_TOKEN
  • Authorization header: Authorization: Bearer YOUR_ADMIN_TOKEN
curl -H "Authorization: Bearer ${ADMIN_TOKEN}" http://localhost:8081/requests
